# **BlaBlaChamb**
## About:
This is just a remake of BlaBlaCar in 2 weeks and a half to train my Symfony skills :)
## Installation:
```
git clone git@github.com:Zen-kun04/BlaBlaChamb.git
cd BlaBlaChamb
cp .env.test .env
composer install
npm i
```
## Server:
1. `symfony serve`
2. `npm run watch`
3. Go to `http://localhost:8000/home`
